# Single-Page HTML CV

Single-Page CV created only with HTML.

## 🛠️ Technologies Used

- **HTML5**
- **Open Graph Protocol**

## 📂 Project Structure

- **Header**: Contact details with active `tel:` and `mailto:` links.
- **Skills**: Core technical competencies (C, C++, Java, Python, etc.).
- **Education**: Academic background.
- **Projects**: Highlighted work with direct GitHub links.
- **Languages**: Spoken languages.
- **Footer**: Important external profiles.

## 🖥️ How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/luismihaistan/single-page-cv.git
   ```
2. **Open the project:**
   Simply open the HTML file in your preferred browser.
